Heading tags- h1 to h6 

hr- horizontal rule
It will divide two elements and make a line between them

br-line break It will move content to new line

pre-Used to display text as we typed.

blockquote- To justify content from both sides

q- To insert quotes

marquee- To animate/scroll text in four directions.

abbr-abbreviation, HTML is an abbreviation of Hyper Text Markup Language.
