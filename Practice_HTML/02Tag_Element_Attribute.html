tag- tag comes in angular-braces<>
        90% tags come with starting tag & ending tag. i.e. h1-h6, p, title, b, strong, i, emp, del, emp etc.
        10% tags are self-closing tags/void tags/empty tags i.e. br, img, hr, inout etc.

        Element- staring tag + content + ending tag

        Attributes- provides additional information which can change the behaviour of our tags.
