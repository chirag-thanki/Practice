Semantic Element

Elements that convey meaning to both the browser and the developer. They are used to describe the content of a web page.
Examples of semantic elements include
  <article>, <aside>, <details>, <figcaption>, <figure>, <footer>, <header>, <main>, <mark>, <nav>, <section>, <summary>
