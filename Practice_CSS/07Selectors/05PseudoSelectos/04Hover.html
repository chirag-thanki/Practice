Hover and Transition!

We're going to hover over div element to change its bg-color and font-color with animation (transition) here.

Transition is nothing but how an animation take place (smooth)

"transition: CSS property (space) time;"

e.g. If we want that the color and width of div increase smoothly then we can write the code as below

"transition: width 5s, background-color 5s;"

We can write as many properties in transition separated by a comma (,)
