align-Items Vs Align-Content

The main difference between align-items and align-content is that: "align-items" is used for single row/column when there is no-wrap of flex.

While align-content is used for multiple-rows/columns with "flex-wrap: wrap;" property.

  • Align-items works for single axis only
  • Align-content works for Multi axis only and along with "flex-wrap" property
  • Align-items take space between them while flex-wrap is ON, While align-content does not take any space except setted "gap property".
